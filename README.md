## 📸 Project Overview

ProjectFlow Dashboard is a modern web-based project management interface designed to help teams visualize project progress, monitor tasks, and track performance metrics.

The dashboard focuses on clean design, usability, and responsive layouts while demonstrating frontend development skills.
